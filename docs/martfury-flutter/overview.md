# Overview

MartFury is a Flutter mobile e-commerce app that connects to a Botble e-commerce backend. One codebase builds Android and iOS.

## Features

- Product browse, search, categories, brands, flash sales
- Authentication: email/password and social login (Google, Facebook, Apple, Twitter)
- Cart, coupons, wishlist, recently viewed
- WebView-based checkout (full payment-gateway compatibility with the Botble backend)
- Order history and tracking
- 29 built-in languages
- Theme: light, dark, system
- Push notifications via Firebase

## Requirements

- A Botble e-commerce website with the API plugin enabled
- Flutter SDK (latest stable)
- Google Play Developer account ($25 one-time) for Android publishing
- Apple Developer account ($99/year) for iOS publishing
