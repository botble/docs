# Support

## Before opening a ticket

1. Check the [FAQ](faq.md).
2. Run through [Troubleshooting](troubleshooting.md).
3. Confirm the same flow works on the website. If it fails on the website, the fix belongs there, not in the app.

## Channels

| Channel | URL | Best for |
|---|---|---|
| Documentation | https://docs.botble.com/martfury-flutter | Setup, configuration |
| Support tickets | https://botble.ticksy.com | Bugs, licensing, technical issues |
| Forum | https://forums.botble.com | Community Q&A |
| Email | contact@botble.com | Business / partnership |

Typical response times: tickets 24–48 hours, email 2–3 business days.

## What to include in a ticket

- The exact error message (screenshot of console or network logs preferred)
- Your website URL
- Steps to reproduce
- What you have already tried
- `flutter --version` output
- Device or emulator and OS version
- Your `.env` file with secrets removed

Do not send passwords, API keys, customer data, or your full Envato purchase code.
