# How to Use Product Gifts

Welcome to the Product Gifts user guide! This section will help you understand how to use the free gift promotion system in your online store.

## What is Product Gifts?

Product Gifts is a promotional system that allows store owners to offer free gift products to customers when they meet certain purchase conditions. Think of it like "Buy X, Get Y Free" promotions.

Customers can:

- **Shop normally** and see gift offers appear automatically
- **Select their preferred gift** when multiple options are available
- **Receive free products** included in their order at $0 cost

## Quick Start

### For Admins

1. Go to your admin panel
2. Click **Ecommerce** > **Product Gifts**
3. Create a new gift rule with minimum order value
4. Add products as gift options
5. Set status to Published

### For Customers

1. Shop and add products to your cart
2. When your cart meets the minimum value, gift options appear
3. Select your preferred gift
4. Complete checkout - gift is included free!

## How Gift Rules Work

```
┌─────────────────────────────────────────────────────────┐
│                    Customer Journey                      │
├─────────────────────────────────────────────────────────┤
│  1. Customer shops and adds products to cart            │
│                         ↓                               │
│  2. Cart value reaches minimum (e.g., $50)              │
│                         ↓                               │
│  3. Gift options automatically appear in cart           │
│                         ↓                               │
│  4. Customer selects preferred gift                     │
│                         ↓                               │
│  5. Customer proceeds to checkout                       │
│                         ↓                               │
│  6. Gift is confirmed and included in order at $0       │
└─────────────────────────────────────────────────────────┘
```

## Gift Tiers Example

You can create multiple gift tiers to encourage larger purchases:

| Cart Value | Gift Tier | Gift Options |
|------------|-----------|--------------|
| $50+ | Bronze | Small accessory |
| $100+ | Silver | Medium product |
| $200+ | Gold | Premium item |

Customers who qualify for multiple tiers get to select one gift from EACH tier!
