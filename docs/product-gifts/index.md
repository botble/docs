# Product Gifts - Free Gifts with Purchase for E-commerce

## Introduction

Product Gifts is a powerful plugin for Botble CMS that allows you to offer free gift products to customers when they meet certain purchase conditions. Boost your sales and customer satisfaction by creating compelling gift promotions that encourage larger orders.

![Gift Rules Management](./images/gift-rules-list.png)

## Features

### Core Features

1. **Flexible Gift Rules**
   - Create unlimited gift promotion rules
   - Set minimum order values to qualify for gifts
   - Schedule promotions with start and end dates
   - Enable/disable rules instantly

2. **Smart Targeting Options**
   - Apply gifts to all products store-wide
   - Target specific products to trigger gifts
   - Target specific product categories
   - Combine multiple conditions

3. **Multiple Gift Items**
   - Add multiple products as gift options per rule
   - Set quantity for each gift item
   - Let customers choose their preferred gift
   - Auto-skip out-of-stock gift items

4. **Beautiful Cart Integration**
   - Gifts appear automatically when customers qualify
   - Multiple display styles (Modern, Minimal, Classic, Elegant)
   - Customizable colors and icons
   - Show/hide product images and original prices

5. **Seamless Checkout Experience**
   - Gift selections persist from cart to checkout
   - Customers can review and change gift choices
   - Gifts show as $0 in order summary
   - Clear gift indicators in admin orders

6. **Admin Management**
   - Easy-to-use admin interface
   - Gift rules table with status indicators
   - Quick edit and delete actions
   - Role-based access permissions

7. **Multi-language Support**
   - Available in 30+ languages
   - English, French, German, Spanish, Vietnamese
   - Japanese, Korean, Chinese, Arabic, and more
   - Easy to add custom translations

8. **Customizable Appearance**
   - Multiple visual styles
   - Custom primary colors
   - Icon selection (emoji support)
   - Show/hide various elements

## Requirements

- Botble CMS version 7.6.4 or higher
- PHP version 8.2 or higher
- Active E-commerce plugin (required)
- MySQL 5.7+ or MariaDB 10.3+
