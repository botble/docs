# Release Notes

## Version 1.0.0 (January 2025)

Initial release of Product Gifts plugin.

### Features

- **Gift Rules Management**
  - Create unlimited gift promotion rules
  - Set minimum order values
  - Schedule with start/end dates
  - Enable/disable rules instantly

- **Smart Targeting**
  - Apply to all products
  - Target specific products
  - Target product categories

- **Multiple Gift Options**
  - Add multiple gift products per rule
  - Set quantity for each gift
  - Customer gift selection

- **Frontend Integration**
  - Product page gift display
  - Cart sidebar integration
  - Checkout gift confirmation
  - Responsive design

- **Customization**
  - Multiple visual styles (Modern, Minimal, Classic, Elegant)
  - Custom colors
  - Emoji icons
  - Show/hide options

- **Multi-language Support**
  - 30+ languages included
  - Easy translation customization

- **Admin Features**
  - Gift rules table with filters
  - Order gift tracking
  - Role-based permissions
  - Settings configuration
