# Release Notes

## Version 1.0.0 - July 7, 2026

### 🎉 Initial Release

We're excited to announce the first release of Velura - Spa, Beauty & Nail Salon Booking Theme for Botble CMS!

### ✨ Features

#### Homepage Variations
- 9 ready-made homepage presets (`Home 1` – `Home 9`)
- One-click install from the CMS installer or via sample data
- Distinct hero styles, section arrangements, and color moods for spas, beauty salons, and nail bars

#### Spa Booking System
- **Service Categories** to organize your treatment menu
- **Services** with pricing, duration, images, and rich descriptions
- **Packages** bundling multiple services at special prices
- **Memberships** with recurring pricing tiers
- **Staff / Specialist Profiles** to showcase your team
- **Appointment Booking** with a modern split-layout reservation form

#### Design & Functionality
- Built on Bootstrap 5.x framework
- 100% responsive design
- Touch-friendly interface
- Elegant typography (Jost body + Rufina headings)
- RTL language support
- Multi-language ready
- SEO optimized structure
- Preloader, scroll animations, and lightbox gallery

#### Content Management
- Blog system for beauty tips and salon news
- Gallery management for treatment and ambiance photos
- Testimonials and client reviews
- Special offers and featured services
- FAQ accordion
- Contact section with map and contact form
- Newsletter integration

#### Technical Features
- Based on Botble CMS (Laravel framework)
- Powerful admin panel
- 15 drag-and-drop UI Block shortcodes
- Widget system (footer, footer bottom, info bar, blog sidebars)
- Custom CSS/JS support
- Google Analytics integration
- Email configuration
- Backup system

### 📋 Requirements
- PHP >= 8.3
- MySQL 5.7+ or MariaDB 10.3+
- Apache, Nginx, or compatible web server
- SSL certificate recommended

## Upgrade Guide

### From Fresh Installation

If this is your first installation of Velura, follow these steps:

1. Download the latest version from CodeCanyon
2. Extract the files to your web server
3. Follow the [installation guide](/velura/installation-web-interface)
4. Import sample data (optional)
5. Configure your spa & booking settings

### Future Updates

When updates are released, you can upgrade by:

1. **Backup your site** - Always create a full backup before updating
2. **Download the update** - Get the latest version from CodeCanyon
3. **Read the changelog** - Check for any breaking changes
4. **Update files** - Replace the updated files
5. **Run migrations** - Execute any database updates
6. **Clear cache** - Clear all application caches
7. **Test thoroughly** - Verify all features work correctly

::: warning Important
Always backup your database and files before performing any updates.
:::
