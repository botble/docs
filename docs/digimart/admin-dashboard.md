# Admin Dashboard

The admin dashboard provides a high-level overview of your marketplace and serves as the entry point for all management tasks.

## Dashboard Overview

After logging in, the dashboard displays key statistics and recent activity:

- **Total Products** — Number of products in the marketplace (all statuses)
- **Total Authors** — Count of seller accounts
- **Total Downloads** — Aggregate download count across all products
- **Pending Approvals** — Products awaiting admin review
- **Recent Activity** — Latest submissions, approvals, and downloads

## Quick Access Sections

### Products & Versions
Manage product listings, approve/reject submissions, view versions and scan status.

**Access:** Sidebar → Products & Versions

**Actions:**
- View all products with current status
- Approve or reject pending products
- Deactivate published products
- Feature products on the homepage
- Manage product versions and dependencies
- View virus scan logs and results

### Authors & Approvals
Manage author accounts and review seller submissions.

**Access:** Sidebar → Authors & Approvals

**Actions:**
- View all author accounts
- Approve pending author registrations
- Ban/unban authors (with reason and duration)
- View author profiles and statistics
- Review author account details

### Categories & Tags
Organize the product taxonomy.

**Access:** Sidebar → Categories & Tags

**Actions:**
- Create and edit product categories (hierarchical)
- Manage category slugs and SEO
- Create and edit product tags
- View products in each category/tag

### Ratings & Reviews
Moderate customer feedback.

**Access:** Sidebar → Ratings & Reviews

**Actions:**
- View all product ratings and reviews
- Approve, reject, or remove reviews
- Respond to reviews as admin or allow author replies
- Track review moderation status

### Product Licensing
Manage licenses and activations for paid products.

**Access:** Sidebar → Product Licensing

**Actions:**
- Create and manage license keys
- View license activation history
- Monitor per-domain license activations
- Deactivate expired or revoked licenses

### Virus Scanning
Monitor and manage product security scans.

**Access:** Sidebar → Virus Scanning

**Actions:**
- View scan logs for all product versions
- Monitor scan status (Pending, Scanning, Clean, Malicious)
- Quarantine malicious files
- View VirusTotal scan reports

### Settings
Configure marketplace behavior and integrations.

**Access:** Sidebar → Settings → Marketplace (tab)

**Actions:**
- Enable/disable license requirements
- Configure virus scanning (VirusTotal API)
- Manage announcements and advertisements
- Set blacklisted domains
- Enable comment notifications
- See [Settings](./settings.md) for complete reference

## Navigation

The admin sidebar provides quick access to all marketplace sections. Use the search bar at the top to quickly navigate to specific pages or manage specific items.

## Statistics & Reports

Hover over dashboard cards to see:
- Trend indicators (up/down from previous period)
- Filtered views by status or author
- Export options (where available)

::: tip
Use the admin dashboard regularly to monitor marketplace health and respond quickly to pending approvals and reported content.
:::
