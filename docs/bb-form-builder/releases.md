# Release Notes

This page contains the release notes for BB Form Builder, detailing the changes and improvements in each version.

## Version 1.1.0 – January 2026

### New Features

- **Autoresponder: Include Submission Details** – New option to automatically send customers a copy of their submitted form data. When enabled, the autoresponder email includes a formatted list of all submitted fields with their values.
