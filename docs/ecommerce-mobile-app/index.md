# Botble Ecommerce Mobile App (React Native + Expo)

A React Native (Expo) mobile e-commerce app that connects to a [Botble e-commerce backend](https://codecanyon.net/user/botble/portfolio). One codebase builds Android and iOS.

## Feature highlights

- Product browse, search, categories, brands, flash sales, recently viewed
- Authentication (email/password, social)
- Cart, coupons, wishlist (server-synced), product comparison
- WebView checkout (compatible with all backend payment gateways)
- Order history, tracking, cancellation, returns
- 15+ built-in languages, RTL support
- Light / dark / system theme

## Tech stack

- React Native + Expo SDK 54
- TypeScript (strict)
- Expo Router v6 (file-based routing)
- React Query + Context API
- NativeWind (Tailwind for React Native)
- react-i18next
