# Theme Options

Theme options allow you to customize your site's look and feel without touching code. You can change colors,
typography, header/footer behavior, and more.

To access the theme options, go to `Appearance` -> `Theme Options` in your admin panel.

![Theme Options](./images/theme-options/theme-option-overview.png)

## General

The **General** tab configures fundamental settings: site name, description, preloader image, and 404 page image.

## Header

### Sticky Header
- Enable/disable sticky header — the header remains visible as the user scrolls.

### Transparent Header
- Toggle a transparent header on the homepage hero section.

### Header Top Bar
- Show or hide the top bar above the main header.
- **Header top text color** — color picker.
- **Header top background color** — color picker.

![Header](./images/theme-options/header.png)

## Footer

### Color Options
- **Background color** — footer background.
- **Background image** — optional image behind footer content.
- **Text color** — body text in the footer.
- **Heading color** — widget titles in the footer.
- **Border color** — divider lines.

### Content Options
- **Footer brand text** — tagline or short description shown next to the logo.
- **Footer service tags** — comma-separated tags displayed below the brand text.
- **Enable decoration images** — toggle decorative background shapes.

![Footer](./images/theme-options/footer.png)

## Styles

### Theme Mode
- **Default theme mode**: System / Light / Dark.
- **Hide theme mode switcher**: toggle whether the dark/light switcher is visible to visitors.

### Visual Effects
- **Enable animation** — scroll and entrance animations.
- **Enable magic cursor** — custom animated cursor on desktop.
- **Enable back to top button** — floating scroll-to-top button.

### Color Scheme
- **Primary color** — main brand color used for buttons, links, and accents.
- **Secondary color** — secondary accent color.
- **Heading color** — `h1`–`h6` color.
- **Body text color** — paragraph and body text color.
- **Link color / Link hover color** — hyperlink states.

![Styles](./images/theme-options/styles.png)

## Breadcrumbs

- **Enable breadcrumb** — Yes / No toggle.
- **Background image** — custom image shown behind the breadcrumb bar.
- **Background color** — fallback color when no image is set.

![Breadcrumbs](./images/theme-options/breadcrumbs.png)

## Typography

The **Typography** settings configure font families and sizes for headings and body text.

- Default font family: **DM Sans**.
- Font sizes are specified in pixels (`px`).
- Adjust these settings to match your design requirements and improve readability.

![Typography](./images/theme-options/typography.png)

## Pages

- **Your homepage displays** — select the page that serves as the site homepage.

## Portfolio (Services)

Configure the style of the service/portfolio detail page.

- **Service style** — Style 1 (sidebar left) or Style 2 (full width).
