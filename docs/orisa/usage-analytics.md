# Google Analytics

Orisa supports Google Analytics integration to track your website traffic directly from the admin panel.

## Setup

1. Go to `Settings` -> `General` in admin panel.
2. Find the **Google Analytics** section.
3. Follow the instructions to connect your Google Analytics account.

## Dashboard Widget

Once connected, analytics data is displayed on the admin dashboard, including:

- Page views
- Sessions
- Bounce rate
- Top pages
- Traffic sources

::: tip
Make sure your Google Analytics property is set up for the correct domain before connecting.
:::
