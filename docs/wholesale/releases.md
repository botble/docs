# Release Notes

## Version 1.0.0 (January 2025)

Initial release of Wholesale plugin.

### Features

- **Customer Groups Management**
  - Create unlimited customer groups
  - Set group-level discounts (percentage or fixed)
  - Configure minimum order quantities and values
  - Priority-based group assignment
  - Published/Draft status control

- **Tiered Pricing Rules**
  - Quantity-based pricing tiers
  - Three discount types: Percentage, Fixed Amount, Fixed Price
  - Min/max quantity range support
  - Product and group-specific rules
  - Automatic price calculation

- **Product MOQ (Minimum Order Quantity)**
  - Set minimum order quantities per product
  - Configure quantity increments
  - Group-specific MOQ settings
  - Automatic quantity validation and adjustment

- **Product Visibility Control**
  - Public, Wholesale Only, and Hidden visibility options
  - Control which products wholesale customers can see
  - Flexible access management

- **Product Group Access**
  - Restrict specific products to specific customer groups
  - Fine-grained access control
  - Multi-group support

- **Wholesale Registration Workflow**
  - Customer application system
  - Admin approval process
  - Application status tracking (pending, approved, rejected)
  - Email notifications for applications

- **Pricing Display**
  - Automatic pricing table on product pages
  - Multiple visual styles (Modern, Minimal, Classic, Elegant)
  - Customizable display options
  - Shows all tiers and savings
  - Mobile-responsive design

- **Settings & Configuration**
  - Enable/disable wholesale system
  - Approval requirements toggle
  - Price visibility control for guests
  - Multiple groups support
  - Discount resolution strategies

- **Admin Features**
  - Customer groups management table
  - Pricing rules management
  - Application review interface
  - Customer assignment management
  - Role-based permissions
  - Settings configuration page

- **Integration**
  - Seamless cart integration
  - Checkout price calculation
  - Order management support
  - Multi-store compatibility
  - Multi-currency support

- **Performance**
  - Optimized database queries
  - Caching support
  - Minimal JavaScript footprint
  - Efficient price calculation
