# Release Notes

## Version 1.0.0 (January 2025)

Initial release.

### Features

- **Customer Groups** - Create unlimited groups with percentage or fixed amount discounts, minimum order requirements, and priority-based resolution
- **Tiered Pricing Rules** - Quantity-based pricing tiers per product with three discount types (Percentage, Fixed Amount, Fixed Price)
- **Product MOQ** - Minimum order quantities and quantity increments per product
- **Product Visibility** - Control product visibility (Public, Wholesale Only, Specific Groups)
- **Wholesale Registration** - Customer application form with admin approval workflow
- **Pricing Table Display** - Automatic pricing table on product pages with four visual styles (Modern, Minimal, Classic, Elegant)
- **Customer Portal** - Wholesale dashboard, company profile editing, reapplication after rejection
- **Settings** - Global enable/disable, approval toggle, guest pricing, multiple groups, discount resolution strategy, display customization
- **Permissions** - 12 granular permissions for role-based access control
- **Integration** - Works with Botble e-commerce cart, checkout, multi-store, and multi-currency
