# Wholesale - B2B E-commerce for Botble CMS

## Introduction

Wholesale is a powerful B2B plugin for Botble CMS that enables you to sell products at different prices to different customer groups, manage minimum order quantities, and control product visibility for wholesale customers. Perfect for businesses selling to both retail and wholesale customers.

![Customer Groups Management](./images/customer-groups-list.png)

## Features

### Core Features

1. **Customer Groups**
   - Create unlimited wholesale customer groups
   - Set discount types (percentage or fixed amount)
   - Configure minimum order quantities and values per group
   - Priority-based group assignment
   - Published/Draft status control

2. **Tiered Pricing Rules**
   - Create quantity-based pricing tiers
   - Three discount types: Percentage, Fixed Amount, Fixed Price
   - Set minimum and maximum quantity ranges
   - Group-specific pricing per product
   - Automatic price calculation

3. **Product MOQ (Minimum Order Quantity)**
   - Set minimum order quantities per product
   - Configure quantity increments (e.g., must order in multiples of 12)
   - Group-specific MOQ settings
   - Automatic quantity validation

4. **Product Visibility Control**
   - Control which products wholesale customers can see
   - Hide products from retail customers
   - Public or group-restricted visibility
   - Flexible access management

5. **Product Group Access**
   - Restrict specific products to specific customer groups
   - Fine-grained access control
   - Multi-group support
   - Automatic filtering

6. **Wholesale Registration Workflow**
   - Customer application system
   - Admin approval process
   - Application status tracking
   - Email notifications

7. **Pricing Table Display**
   - Automatic pricing table on product pages
   - Shows all quantity tiers and discounts
   - Customizable display styles
   - Mobile-responsive design

8. **Seamless Integration**
   - Works with existing Botble e-commerce
   - Transparent price calculation
   - Cart and checkout integration
   - Order management support

## Requirements

- Botble CMS version 7.6.0 or higher
- PHP version 8.2 or higher
- Active E-commerce plugin (required)
- MySQL 5.7+ or MariaDB 10.3+
