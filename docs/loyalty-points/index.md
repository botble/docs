# Loyalty Points - Advanced Loyalty Program for E-commerce

## Introduction

Loyalty Points is a comprehensive loyalty program and rewards system designed for e-commerce stores built with Botble CMS. It enables store owners to create and manage a powerful loyalty program where customers earn points from purchases and various activities, then redeem them for discounts at checkout. The plugin provides a complete solution for increasing customer retention, boosting repeat purchases, and building long-term customer relationships.

## Features

### Core Features

1. **Points Earning System**
   - Customers earn points from purchases
   - Configurable points-to-currency exchange rate
   - Flexible earning rates (points per amount spent)
   - Order status-based point awards
   - Automatic point calculation at checkout

2. **Points Redemption**
   - Redeem points for discounts at checkout
   - Configurable redemption rates
   - Minimum and maximum redemption limits
   - Maximum discount percentage cap
   - Real-time discount calculation

3. **Member Levels/Tiers**
   - Create custom membership tiers (Bronze, Silver, Gold, Platinum, etc.)
   - Points-based tier progression
   - Tier-specific earning multipliers
   - Custom benefits per tier
   - Automatic tier upgrades based on lifetime points

4. **Bonus Points Activities**
   - Registration bonus points for new customers
   - Product review bonus points
   - Photo review bonus points (extra rewards)
   - Referral bonus points
   - Birthday bonus points

5. **Points Expiry Management**
   - Configurable expiration period
   - Automatic expiry processing
   - Customer notifications before expiry
   - Transaction history for expired points

6. **Customer Dashboard**
   - View current points balance
   - Track lifetime points earned
   - See current membership tier and benefits
   - Progress indicator to next tier
   - Complete transaction history
   - Points earning and redemption guide

7. **Admin Management**
   - Customer points overview
   - Manual point adjustments (add/deduct)
   - Transaction history viewing
   - Point adjustment notes
   - Bulk operations support

8. **Checkout Integration**
   - Seamless points redemption at checkout
   - Display points to be earned
   - Quick redemption percentage buttons
   - Real-time validation
   - Points shown on order confirmation

9. **Product Page Integration**
   - Display potential points earnings on products
   - Show maximum discount available with points
   - Encourage purchases with points information

10. **Reports & Analytics**
    - Total points in circulation
    - Redemption rate tracking
    - Top customers by points
    - Activity trends over time
    - Points earned vs redeemed charts

11. **Order Integration**
    - Automatic points award on eligible orders
    - Points reversal for cancelled orders
    - Points information on order detail pages
    - Admin order view with points data

12. **Multi-language Support**
    - Available in 30+ languages
    - Easy translation from admin panel
    - Supports RTL languages
    - Localized number and currency formats

## Requirements

- Botble CMS version 7.5.0 or higher
- PHP version 8.2 or higher
- Active E-commerce plugin (required)
- MySQL 5.7+ or MariaDB 10.3+
