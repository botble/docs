# Upgrading Loyalty Points

This guide explains how to upgrade your Loyalty Points plugin to the latest version.

## Before You Begin

**Important:** Always create a backup of your website before upgrading. Contact your hosting provider if you need help with backups.

## Upgrade Steps

If your license is already activated:

1. Go to **Admin** > **Plugins** > **Installed plugins**
2. Find Loyalty Points in the list
3. Click the **Update** button to download and install the latest version
4. Go to **Admin** > **Platform Administration** > **Cache management**
5. Click "Clear all CMS cache"

## After Upgrading

Verify everything works correctly:

- Check customer points balances are displaying correctly
- Test points earning on a test order
- Test points redemption at checkout
- Verify member levels are assigned properly

## Need Help?

If you encounter any issues after upgrading:

1. Clear your browser cache and try again
2. Check the [Troubleshooting guide](/loyalty-points/troubleshooting)
3. Contact our support team with details about the issue

**When contacting support, please include:**
- Your current plugin version
- Your Botble CMS version
- Description of the issue
- Any error messages you see
