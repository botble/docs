# Installing Loyalty Points

This guide will walk you through the process of installing the Loyalty Points plugin for your Botble E-commerce store.

## Requirements

Before installing Loyalty Points, ensure your system meets the following requirements:

- Botble CMS version 7.5.0 or higher
- PHP version 8.2 or higher
- MySQL 5.7+ or MariaDB 10.3+
- Botble E-commerce plugin (required)

::: warning Important
This plugin **requires the E-commerce plugin** to be installed and activated. Loyalty Points is designed to work **only with Botble e-commerce scripts**.
:::

## Installation Steps

1. Download the plugin from [CodeCanyon downloads page](https://codecanyon.net/downloads)
2. Extract the downloaded zip file
3. Upload the extracted folder to `platform/plugins/loyalty-points` directory
4. Go to **Admin** > **Plugins** and activate the Loyalty Points plugin
5. Refresh your website

## Post-Installation Setup

After installing and activating the Loyalty Points plugin:

### 1. Activate Your License

- Navigate to **Loyalty Points** > **License Activation**
- Enter your purchase code from CodeCanyon
- Click "Activate license"

### 2. Access Admin Interface

- The **Loyalty Points** menu item will appear in your admin panel
- Click on **Loyalty Points** to access the management system

### 3. Configure Basic Settings

- Navigate to **Settings** > **Loyalty Points**
- Set up your points exchange rate
- Configure earning and redemption rates
- Set up bonus points for activities

### 4. Create Member Levels (Optional)

- Go to **Loyalty Points** > **Member Levels**
- Click **Create** to add membership tiers
- Set point thresholds and earning multipliers

## Verifying Installation

To verify that the plugin has been installed correctly:

1. Log in to your admin panel
2. Verify the **Loyalty Points** menu item appears in the main navigation
3. Click on **Loyalty Points** to access the management interface
4. Check that all sub-menus are accessible (Customer Points, Transactions, Member Levels, Reports)
5. Go to **Settings** > **Loyalty Points** and verify settings are accessible
