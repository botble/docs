# Release Notes

## Version 1.0.0 (March 2026)

Initial release.

### Features

- **Preorder Products** — Configure preorder pricing per product with three strategies (Full Price, Deposit Percentage, Deposit Fixed)
- **Order Lifecycle** — 9-stage order tracking (Requested → Accepted → Prepayment → Final Pre-Order → Shipping → Delivered + Cancelled/Refunded)
- **Two-Stage Payment** — Deposit collection at checkout, balance payment when product ships. Supports COD, bank transfer, Stripe, PayPal
- **Preorder Discounts** — Exclusive percentage or fixed amount discounts for preorder customers
- **Stock Control** — Stock limits and minimum purchase quantities per product
- **Customer Dashboard** — Self-service preorder management with payment, cancellation, and refund request
- **Vendor Support** — Vendor preorder management when Marketplace plugin is active
- **Auto-Expiration** — Daily command disables preorder products past their availability date
- **Display Options** — Configurable badge color and 4 display styles (Minimal, Banner, Inline, Detailed)
- **Mixed Cart** — Configurable setting to allow or deny mixing regular and preorder items
- **Admin Dashboard** — Aggregated statistics, status breakdown, and sales data
- **Permissions** — 9 granular permissions for role-based access control
- **Multi-language** — Translations for 42+ locales
