# Product Variations

Product variations allow you to sell products with different attributes such as size, color, or material.

## Creating Variations

1. In admin panel, go to `Ecommerce` -> `Products`
2. Edit a product or create a new one
3. In the **Variations** tab, click `Add new variation`
4. Define variation attributes (e.g., Size: S, M, L; Color: Red, Blue)
5. Set price, stock, and SKU for each variation
6. Click `Save`

## Variation Display

On the product page, customers see variation options as selectable buttons. When a variation is selected:

- Price updates in real-time
- Stock availability changes
- Product image updates (if variation-specific images are set)

## Managing Stock

Each variation has its own stock quantity. When a variation is out of stock, it appears as disabled on the product page.

::: tip
Upload unique images for each variation to help customers visualize their selection, especially for color variants.
:::
