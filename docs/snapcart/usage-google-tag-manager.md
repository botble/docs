# Google Tag Manager

Integrate Google Tag Manager (GTM) for advanced tracking and marketing tag management.

## Setup

1. Create a GTM account at [tagmanager.google.com](https://tagmanager.google.com)
2. Create a new container for your website
3. In admin panel, go to `Settings` -> `General`
4. Enter your GTM Container ID (e.g., `GTM-XXXXXX`)
5. Click `Save`

## Usage

Once configured, GTM code is automatically injected into all pages. You can then manage tags for:

- Google Analytics 4
- Facebook Pixel
- Conversion tracking
- Remarketing
- Custom events

::: tip
Use GTM's built-in ecommerce data layer to track add-to-cart, checkout, and purchase events automatically.
:::
