# Translation

The Translation plugin allows you to translate the theme's interface strings (buttons, labels, messages) into any language.

## Setup

1. In admin panel, go to `Plugins` -> `Installed Plugins`
2. Activate the **Translation** plugin
3. Go to `Settings` -> `Localization` -> `Theme Translations`

## Translating Theme Strings

1. Go to `Settings` -> `Localization` -> `Theme Translations`
2. Select the language you want to translate to
3. You will see a list of all translatable strings with their English values
4. Enter the translations for each string
5. Click `Save`

## Translating Admin Panel

1. Go to `Settings` -> `Localization` -> `Other Translations`
2. Select the group and language
3. Enter translations and save

::: tip
SnapCart includes 24 language files out of the box. Check if your language is already partially translated before
starting from scratch.
:::
