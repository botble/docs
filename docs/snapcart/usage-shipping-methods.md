# Shipping Methods

Configure shipping options and rates for your store.

## Managing Shipping Methods

Go to `Ecommerce` -> `Shipping` in the admin panel.

## Available Methods

### Flat Rate Shipping

Set a fixed shipping fee for all orders or configure rates by location.

1. Click `Add shipping rule`
2. Set the shipping name and price
3. Configure location-based rates if needed
4. Click `Save`

### Free Shipping

Offer free shipping based on minimum order amount.

1. Add a shipping rule with price set to `0`
2. Set the minimum order condition

### Shipping by Location

Configure different shipping rates based on customer location. See [Shipping by Location](./usage-shipping-by-location.md) for details.

### Shippo Integration

Integrate with Shippo for real-time carrier rates and label printing. See [Shippo Integration](./usage-shippo.md) for setup instructions.

::: tip
SnapCart's delivery time picker works alongside shipping methods. Configure both for a complete delivery experience.
:::
