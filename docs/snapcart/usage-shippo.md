# Shippo Integration

Integrate with Shippo for real-time shipping rates, label printing, and tracking.

## Prerequisites

- A Shippo account at [goshippo.com](https://goshippo.com)
- Shippo API key

## Setup

1. In admin panel, go to `Plugins` -> `Installed Plugins` and activate **Shippo**
2. Go to `Ecommerce` -> `Settings` -> `Shipping` -> `Shippo`
3. Enter your Shippo API key
4. Configure your sender address (warehouse/store address)
5. Click `Save`

## Features

- **Real-time Rates**: Show live carrier rates at checkout (USPS, FedEx, UPS, DHL, etc.)
- **Label Printing**: Generate and print shipping labels from the admin panel
- **Tracking**: Automatic tracking number generation and customer notifications

## Checkout Experience

When Shippo is enabled, customers see real-time shipping options with prices from multiple carriers at checkout.

::: tip
Set up your product weights and dimensions accurately for precise shipping rate calculations.
:::
