# Product Import/Export

Bulk manage products by importing from or exporting to spreadsheet files.

## Exporting Products

1. In admin panel, go to `Ecommerce` -> `Products`
2. Click the `Export` button
3. Select the export format (CSV or Excel)
4. The file will be downloaded automatically

## Importing Products

1. Go to `Ecommerce` -> `Products`
2. Click the `Import` button
3. Download the sample template first to see the required format
4. Fill in your product data following the template structure
5. Upload the completed file
6. Review the import preview
7. Click `Import` to confirm

## Import Fields

The import template includes fields for:

- Product name, description, price
- SKU, stock quantity
- Categories, tags, brand
- Images (URL format)
- Variations and attributes

::: warning
Always back up your database before importing large product files. Test with a small batch first.
:::
