# Google Analytics

Track your store's traffic and customer behavior with Google Analytics integration.

## Setup

1. In admin panel, go to `Plugins` -> `Installed Plugins` and activate the **Analytics** plugin
2. Go to `Settings` -> `General` -> `Google Analytics`
3. Follow the setup instructions to connect your Google Analytics account

## Dashboard Widget

Once configured, Google Analytics data will be displayed in your admin dashboard, showing:

- Page views
- Sessions
- Bounce rate
- Traffic sources

::: tip
Use Google Analytics alongside the [Google Tag Manager](./usage-google-tag-manager.md) for advanced tracking such as
ecommerce events, conversion tracking, and remarketing.
:::
