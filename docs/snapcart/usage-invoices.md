# Invoices

Generate and manage invoices for customer orders.

## Viewing Invoices

Invoices are automatically generated for each completed order. Go to `Ecommerce` -> `Invoices` in the admin panel.

## Invoice Template

Customize the invoice template in `Ecommerce` -> `Settings` -> `Invoice`. You can modify:

- Company information (name, address, phone, email)
- Logo
- Tax information
- Footer text

See [Invoice Template](./invoice-template.md) for advanced customization.

## Downloading Invoices

- **Admin**: Click the download button on any order or invoice page
- **Customer**: Customers can download invoices from their order history in their account dashboard
