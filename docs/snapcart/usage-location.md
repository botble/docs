# Location

Configure countries, states, and cities for shipping, tax, and customer address management.

## Setup

1. In admin panel, go to `Plugins` -> `Installed Plugins` and activate **Location**
2. Go to `Settings` -> `Location`

## Importing Location Data

1. Go to `Settings` -> `Location`
2. Click `Import location data`
3. Select the countries you want to import
4. Click `Import`

This imports countries, states/provinces, and cities with their data.

## Usage

Location data is used for:

- **Shipping**: Calculate shipping rates by location
- **Tax**: Apply tax rules based on customer location
- **Customer addresses**: Provide dropdowns for country, state, and city selection
- **Store configuration**: Set your store's default country

::: tip
Import only the countries you ship to. This keeps the location database lean and improves address form performance.
:::
