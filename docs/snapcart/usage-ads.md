# Ads

Display promotional banners and advertisements across your store.

## Managing Ads

Go to `Ads` in the admin panel.

## Creating an Ad

1. Click `Create`
2. Fill in the details:

- **Name**: Ad identifier
- **Image**: The banner image
- **URL**: Destination link when clicked
- **Location**: Where to display the ad (header, sidebar, footer, etc.)
- **Expired At**: When the ad should stop displaying

3. Click `Save`

## Ad Locations

SnapCart supports displaying ads in various positions through the theme's shortcode system and widget areas.

::: tip
Use high-quality, mobile-optimized images for ads. Keep file sizes small for fast loading on mobile connections.
:::
