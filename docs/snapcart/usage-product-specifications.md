# Product Specifications

Display detailed product specifications in a structured table format on product pages.

## Adding Specifications

1. In admin panel, go to `Ecommerce` -> `Products`
2. Edit a product
3. In the **Specification** tab, add specification groups and attributes
4. Fill in specification name and value pairs (e.g., Weight: 500g, Material: Cotton)
5. Click `Save`

## Specification Groups

Organize specifications into groups for better readability:

- **General**: Brand, model, weight, dimensions
- **Technical**: Material, power, capacity
- **Custom**: Any product-specific attributes

::: tip
Detailed specifications help customers make informed purchase decisions and reduce return rates.
:::
