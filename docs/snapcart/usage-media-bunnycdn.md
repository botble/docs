# Media - Setup BunnyCDN

Store and deliver media files using BunnyCDN for fast global content delivery.

## Prerequisites

- A BunnyCDN account
- A storage zone created in BunnyCDN

## Configuration

1. In admin panel, go to `Settings` -> `Media`
2. Select **BunnyCDN** as the storage driver
3. Fill in the credentials:

- **Storage Zone Name**: Your BunnyCDN storage zone name
- **API Key**: Your storage zone API key
- **Region**: Storage zone region
- **Pull Zone URL**: Your pull zone URL for serving files

4. Click `Save`

::: tip
BunnyCDN offers excellent performance-to-cost ratio, making it ideal for image-heavy ecommerce stores.
:::
