# Discounts & Coupons

Create discount campaigns and coupon codes to boost sales and reward customers.

## Creating a Discount

1. In admin panel, go to `Ecommerce` -> `Discounts`
2. Click `Create`
3. Fill in the discount details:

- **Type**: Percentage or Fixed Amount
- **Value**: The discount amount or percentage
- **Code**: Coupon code customers enter at checkout (leave empty for automatic discounts)
- **Quantity**: Maximum number of times the discount can be used
- **Start Date / End Date**: The promotion period
- **Minimum Order Amount**: Minimum cart value required

4. Click `Save`

## Discount Types

### Coupon Code

Customers enter a code at checkout to receive the discount. Useful for marketing campaigns and promotions.

### Automatic Discount

Applied automatically when conditions are met (e.g., minimum order amount). No code required.

## Managing Discounts

View all discounts and their usage statistics in `Ecommerce` -> `Discounts`. You can edit, disable, or delete
discounts at any time.

::: tip
Combine discounts with [Flash Sales](./usage-flash-sales.md) for maximum promotional impact.
:::
