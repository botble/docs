# Product Options

Product options allow customers to customize products with additional choices that may affect pricing.

## Creating Product Options

1. In admin panel, go to `Ecommerce` -> `Product Options`
2. Click `Create` to add a new option set
3. Define option type (dropdown, radio, checkbox, text field)
4. Set option values with optional price adjustments
5. Click `Save`

## Assigning Options to Products

1. Edit a product in `Ecommerce` -> `Products`
2. Go to the **Options** tab
3. Select the option sets to apply
4. Click `Save`

## Option Types

- **Dropdown**: Single selection from a list
- **Radio buttons**: Single selection displayed as radio buttons
- **Checkbox**: Multiple selections allowed
- **Text field**: Free text input from customers

::: tip
Use product options for customizations like gift wrapping, engraving, or custom sizing that apply across multiple products.
:::
