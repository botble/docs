# Currencies

Configure multiple currencies to serve international customers.

## Managing Currencies

Go to `Ecommerce` -> `Settings` -> `Currencies` in the admin panel.

## Adding a Currency

1. Click `Add new currency`
2. Fill in the details:

- **Title**: Currency name (e.g., US Dollar)
- **Symbol**: Currency symbol (e.g., $)
- **Decimals**: Number of decimal places
- **Exchange Rate**: Rate relative to the default currency
- **Is Prefix Symbol**: Display symbol before or after the amount

3. Click `Save`

## Default Currency

Set one currency as the default. All product prices are stored in the default currency and converted based
on exchange rates.

## Currency Switcher

SnapCart includes a currency selector in the customer account area. Customers can switch currencies from the
language/currency modal accessible via the bottom navigation.
