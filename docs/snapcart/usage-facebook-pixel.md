# Facebook Pixel

Track customer behavior and optimize Facebook/Meta advertising with the Facebook Pixel integration.

## Setup

1. Create a Facebook Pixel in your [Meta Business Suite](https://business.facebook.com)
2. In admin panel, go to `Settings` -> `General`
3. Enter your Facebook Pixel ID
4. Click `Save`

## Tracked Events

The Facebook Pixel automatically tracks:

- **PageView**: Every page visit
- **ViewContent**: Product page views
- **AddToCart**: Items added to cart
- **InitiateCheckout**: Checkout started
- **Purchase**: Completed orders

## Usage

Use tracked data in Meta Ads Manager to:

- Create custom audiences from store visitors
- Build lookalike audiences for prospecting
- Optimize ad delivery for conversions
- Retarget cart abandoners
