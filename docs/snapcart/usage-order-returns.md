# Order Returns

Allow customers to request returns for their orders.

## Enabling Order Returns

Order returns are available by default in the ecommerce plugin. Customers can submit return requests from their
account dashboard.

## Processing Returns

1. In admin panel, go to `Ecommerce` -> `Order Returns`
2. Review the return request details
3. Approve or reject the return
4. Process the refund if applicable

## Return Request Details

Each return request includes:
- Order reference number
- Products to return
- Reason for return
- Customer notes

::: tip
A clear and fair return policy builds customer confidence and can increase purchase rates, especially for first-time
buyers on mobile.
:::
