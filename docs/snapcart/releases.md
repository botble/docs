# Release Notes

## 1.0.0 (2026-02-03)

- Initial release
- Mobile-focused design optimized for 480px viewports
- Bottom navigation with 5 tabs (Home, Categories, Cart, Account, More)
- Floating contact buttons (Zalo, Phone, Messenger)
- AJAX cart with modal popup
- Sticky footer cart bar
- One-page checkout with delivery time picker
- Product reviews and social proof badges
- 8 homepage shortcodes (hero carousel, featured products, categories, flash sale, reviews, brands, blog posts)
- Multiple payment gateway support (PayPal, Stripe, Paystack, Razorpay, Mollie, SSLCommerz)
- RTL language support
- Lazy loading images
- Accessibility improvements
