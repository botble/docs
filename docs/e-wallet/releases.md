# What's New

See what's been added, changed, or fixed in each version of E-Wallet.

## Version 1.0.0

**Released:** December 2025

This is the first release of E-Wallet!

### What's Included

#### For Your Customers

| Feature | Description |
|---------|-------------|
| **My Wallet Page** | Customers can see their balance and history |
| **Add Money** | Customers can top-up using credit card, PayPal, etc. |
| **Pay with Wallet** | Use wallet balance at checkout |
| **Get Refunds** | Refunds go directly to wallet |
| **Withdraw Money** | Request to transfer money to bank or PayPal |

#### For Store Admins

| Feature | Description |
|---------|-------------|
| **Dashboard** | See wallet statistics at a glance |
| **Manage Wallets** | View all customer wallets |
| **Adjust Balances** | Add or remove money from wallets |
| **Handle Top-ups** | Complete or cancel deposit requests |
| **Handle Withdrawals** | Approve or reject withdrawal requests |
| **View Transactions** | See all wallet activity |
| **Settings** | Configure limits, payment methods, and more |

#### Other Features

- Works in 30+ languages
- Secure transactions
- Complete transaction history
- Role-based admin access

### Requirements

To use E-Wallet, you need:

| Requirement | Version |
|-------------|---------|
| Botble CMS | 7.5.0 or newer |
| E-commerce Plugin | Must be installed |
| PHP | 8.2 or newer |
