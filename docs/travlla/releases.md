# Release Notes

This page contains the release notes for Travlla, detailing the changes and improvements in each version.

## Version 1.0.0 – June 25, 2026

### Initial Release
- First public release of **Travlla - Travel & Tour Booking Laravel System**, built on Botble CMS.
- 5 ready-made homepage layouts.
- Tour Manager: tours, tour packages, destinations, tour categories, pricing plans, and discounts.
- Tour booking & checkout flow with customer support.
- Galleries, blog, contact, newsletter, and pages modules.
- Travel-focused UI blocks (shortcodes): popular tours, popular destinations, tour categories, offers & deals, testimonials, and more.
- Multi-language and full RTL support.
- SEO & sitemap support, Google Analytics integration.
- Media system with Amazon S3, DigitalOcean Spaces, BunnyCDN, and Wasabi support.
- Fully responsive, PSR-12 compliant codebase.
