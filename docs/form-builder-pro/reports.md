# Reports and Analytics

Track your form performance with built-in analytics and reports.

## Accessing Reports

Navigate to **Form Builder Pro** > **Reports** to view:

### Overview Statistics

- Total forms count
- Total submissions count
- Active forms count

### Time-Based Metrics

- Today's submissions
- This week's submissions
- This month's submissions

### Submissions Chart

Visual graph of submissions over the last 30 days

### Top Performing Forms

Forms with the most submissions

## Individual Form Statistics

Each form shows:

- Total submission count
- Submissions today
- Submissions this week
- Submissions this month
- Field count

## Using Reports for Optimization

1. **Identify Popular Forms**: Focus on improving high-traffic forms
2. **Track Trends**: Monitor submission patterns over time
3. **Form Performance**: Compare forms to identify best practices
4. **Conversion Tracking**: Analyze submission rates
