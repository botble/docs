# Form Builder Pro - Advanced Form Builder for Botble CMS

## Introduction

Form Builder Pro is a powerful, feature-rich form builder plugin designed for websites built with Botble CMS. It enables website owners to create beautiful, responsive forms with an intuitive drag-and-drop interface. From simple contact forms to complex multi-step wizards, Form Builder Pro provides all the tools you need to collect data from your visitors effectively.

## Compatibility

Form Builder Pro is fully compatible with all scripts based on Botble CMS, including all products available on our [CodeCanyon portfolio](https://codecanyon.net/user/botble/portfolio). This includes but is not limited to:

- Farmart - Laravel Ecommerce
- Martfury - Laravel Ecommerce
- Shofy - Laravel Ecommerce
- JobBox - Laravel Job Portal
- And all other Botble-based products

## Features

### Core Features

1. **Drag & Drop Form Builder**
   - Intuitive visual form builder interface
   - Real-time form preview
   - Drag and drop field arrangement
   - Easy field configuration
   - Multiple field types support

2. **Rich Field Types**
   - Text input fields (text, number, email, phone)
   - Textarea for multi-line text
   - Autocomplete field
   - Select dropdowns and radio buttons
   - Checkboxes
   - Date and time pickers
   - File and image uploads
   - Hidden fields
   - HTML content blocks

3. **Multi-Step Forms (Wizard)**
   - Create multi-step form wizards
   - Horizontal or vertical step layout
   - Progress indicators
   - Step validation
   - Navigation between steps

4. **Form Styling & Customization**
   - Multiple pre-built form styles (Default, Modern, Minimal, Bold, Glassmorphism)
   - Customizable colors and appearance
   - Adjustable form sizes and alignment
   - Border radius and shadow options
   - Responsive design for all devices

5. **Popup Forms**
   - Display forms in modal popups
   - Multiple trigger options:
     - Button click
     - Time delay
     - Scroll percentage
     - Exit intent
   - Customizable popup appearance

6. **Form Actions & Integrations**
   - Email notifications with customizable templates
   - Database storage for submissions
   - API webhook integration
   - Mailchimp integration for email marketing
   - GetResponse integration for campaigns

7. **Submission Management**
   - View all form submissions in admin panel
   - Filter submissions by form
   - Export submissions to CSV/Excel
   - Download uploaded files
   - Submission analytics and reports

8. **Security Features**
   - Google reCAPTCHA integration
   - Math CAPTCHA option
   - File upload validation
   - XSS and injection protection
   - CSRF protection

9. **Embedding Options**
   - Shortcode embedding `[form-builder-pro code="..."][/form-builder-pro]`
   - Direct URL embedding
   - JavaScript embed for external sites
   - iframe support

10. **Reporting & Analytics**
    - Form submission statistics
    - Daily/weekly/monthly reports
    - Top performing forms
    - Submission trends visualization

11. **Multi-language Support**
    - Complete translation support
    - Available in multiple languages
    - Easy to add new translations
    - Localized form labels and messages

## Requirements

- Botble CMS version 7.5.0 or higher
- PHP version 8.2 or higher
- MySQL 5.7+ or MariaDB 10.3+
